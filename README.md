# Web-Controlled ESP32 Servo

這是一個利用 ESP32 控制伺服馬達的專案，支援「手機網頁」與「實體紅外線遙控器」雙重操作。

為了解決傳統網頁伺服器跟馬達 PWM 訊號互相干擾的問題，這裡寫了 FreeRTOS 雙核心來分工，讓網頁上的儀表板跟實體馬達可以做到滿順的即時同步。

## 硬體需求
* ESP32 開發板
* SG90 伺服馬達
* VS1838B 紅外線接收器

## 功能特點
* **數位孿生網頁端**：用 WebSocket 寫的，手機點按鈕的瞬間實體馬達就會動，網頁指針也會同步。
* **FreeRTOS 雙核心**：Core 0 專門跑 WiFi 跟網頁伺服器，Core 1 負責算馬達角度跟接紅外線訊號，兩邊互不卡頓。
* **正弦波緩起停**：捨棄了線性的角度加減，用 cos 函數算軌跡，馬達轉起來比較順，不會有那種快把齒輪弄壞的抖動感。
